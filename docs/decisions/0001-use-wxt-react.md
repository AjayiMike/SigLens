# 0001: Use WXT + React

Status: accepted

## Context

SigLens requires a browser extension with popup/options/background entrypoints, strict typing, and fast iteration.

## Decision

Use WXT with React + TypeScript.

## Consequences

- Manifest is generated by WXT config and entrypoints.
- Browser APIs are accessible from extension contexts with predictable build/runtime behavior.
- The project keeps one package/repository for v1 simplicity.
